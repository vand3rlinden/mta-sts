MTA-STS Policy

vand3rlinden.com enforces MTA-STS (Mail Transfer Agent Strict Transport Security) to enhance the security of email communication by requiring SMTP connections to use TLS (Transport Layer Security).

Current MTA-STS policy for vand3rlinden.com: http://mta-sts.vand3rlinden.com/.well-known/mta-sts.txt.

Deployment: Host the MTA-STS Policy on Github Pages.

For more information, please refer to the MTA-STS RFC.
